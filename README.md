## Usage

This module requires to be bundled with [webpack][webpack]/browserify and loads `react/addons` internally. WIP: release a UMD version via bower/components.


### Development

```shell
git clone ...
npm i
npm run dev
```

Then open [http://localhost:8080](http://localhost:8080).
